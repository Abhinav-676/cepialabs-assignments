# Why do we use React?
We use react because it allows us to manage a website more efficiently by able to divide a website into components,
There Components are can be tested and developed individually, which makes it easier to focus our developmental work.
React also handles DOM manipulation by itself, allowing us to focus of the core business logic and the UI.

# What is Vite? Why is it better than create-react-app?
Vite is a development toolkit, it is desigded to provide with the tools and dependencies required for a framework like react.
We use Vite because it is modern, comes with latest features and is easy to use. 
VIte also is in active development while, create-react-app has been discontinued.
For these reasons it is better to use Vite for creating a React Application than create-react-app.
